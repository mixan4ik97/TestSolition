Задание

1. Собрать проект, разобраться в нём (img_1.jpg - схема работы проекта). 
Суть проекта 
- принять данные. данные могут быть в разных форматах
- если данные могут быть обработаны сразу - отдать в умножитель
- если данные не являются строкой из цифр - преобразовать так чтоб получилась строка из цифр
- каждую цифру умножить на 2
- результат умножение если нужно - закодировать обратно
- вывести данные

2. Добавить класс чтения исходных данных из файла, читать данные из файла когда запускаем программу 
 с параметром (с именем файла). Выводить результаты всё равно в консоль.
 
3. Добавить класс декодера входных данных из Base64. Встроить в программу. Считать что данные лежат в Base64 если первый символ данных "=".
Формат входных данных {'=', 'данные в base64'}. 
Примеры - "=MTIzNDU=" (закодировано 12345), "=MTIzNDU2" (закодировано 123456)
Считать что данные, закодированные через Base64, содержат только цифры.
Свой декодер Base64 писать не надо, взять любой из интернета.
Энкодер (переводить результат после умножения на 2 обратно в Base64) делать не надо.

4. (один из вариантов на выбор)
- Результат должен собираться через cmake (если знаком cmake)
- Результат содержит свой скрипт для сборки (например запускающий gcc)
- Результат содержит файл проекта для сборки (например от Visual Studio)

5. Быть готовым рассказать о своём решении и обсудить проект

Оценивается
1 Компилируется-нет, возможность кросскомпиляции 
2 Принятые решения по встраиваванию нового кода
3 Если решение проекта выглядит очевидным - то можно не писать код, но быть готовым рассказать как реализовывать задание
